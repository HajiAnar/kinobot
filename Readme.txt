Kinopoisk Movie Bot

Этот бот интегрируется с API Kinopoisk для поиска фильмов по жанру, количеству и рейтингу.
Это позволяет пользователям быстро и легко находить фильмы, которые их интересуют.

Описание работы
Бот выполняет следующие задачи:

Поиск фильмов:
осуществляет поиск фильмов на основании указанных пользователем критериев:
жанр, количество фильмов, рейтинг и тип сортировки.
Формирование запроса: использует API Kinopoisk для получения информации о фильмах.
Возвращает информацию: предоставленный список фильмов включает в себя различные детали,
такие как название, рейтинг и жанр.

Основной процесс:
Пользователь вводит желаемый жанр, количество фильмов для получения, и опции для сортировки.
Бот формирует запрос к API Kinopoisk с использованием этих параметров.
Бот обрабатывает ответ API и возвращает пользователю список фильмов.

Команды: /low/high/customs/history

Инструкция по установке и запуску

Требования
Перед началом убедитесь, что на вашей системе установлены следующие компоненты:

Python 3.x
Установленный модуль requests

Склонируйте репозиторий:
bash
   git clone <9fe6dd07ba909a427078960ff3da0bb074feb833>
   cd <название папки>
Установите необходимые зависимости:
Убедитесь, что requests установлен:
bash
   pip install requests
Настройка API ключа:
Зарегистрируйтесь на https://kinopoisk.dev/ для получения API ключа и замените значение API_KEY в коде.

Запустите скрипт:
bash
Копировать
   python <имя_скрипта>.py
Следуйте инструкциям в терминале для ввода необходимых параметров.
Пример использования
Запустите скрипт и введите параметры по запросу:

Жанр: например, комедия
Количество фильмов: например, 5

Бот возвращает список фильмов с указанием выбранных параметров.

Примечания
Перед использованием убедитесь, что соблюдаются условия и лимиты API Kinopoisk.
